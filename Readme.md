# Antd-Access 后端项目

## 前期目标
其实一开始的时候呢,可以当做是另一个项目来看.  
后端的搭建, 可以这么理解:   
1. 最初目标是使用 Spring Security 来搭建一个用户登录(不涉及数据库,使用内存用户的方式).
2. 在用户登录完成后, 再来将数据库和用户关联起来.
3. 然后就是角色以及菜单的数据结构.
4. 然后就是用户和角色的关联.
5. 前后端联调.
